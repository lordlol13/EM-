3. Подставляем под формулу: $P(t)=1−e^{−\lambda * t}$ (e - постоянная)

$$0.95 = 1 - e^{-30\lambda} \implies e^{-30\lambda} = 0.05$$
$$-30\lambda = \ln(0.05) \implies \lambda \approx 0.0999$$

$$P(10) = 1 - e^{-10 \cdot 0.0999} = 1 - e^{-0.999} \approx 0.632$$

$$P(27) = 1 - e^{-27 \cdot 0.0999} = 1 - e^{-2.697} \approx 0.933$$

**Ответ:** 63,2% и 93,3%
